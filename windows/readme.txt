           HIGGSFIELD AI - VIDEO FILTER

--- QUICK START ---

1. Open a terminal (CMD or PowerShell) in this folder.
2. Run: start.bat
3. Follow the on-screen prompts to apply AI filters.

--- NOTES ---

* First run automatically verifies hardware and installs dependencies.
* Requires NVIDIA GPU with 12GB+ VRAM for optimal performance.
* Supported weights (ltx-2-19b-distilled.safetensors) must be in src/.
* Filtered videos save to the root directory as 'filtered_*.mp4'.
